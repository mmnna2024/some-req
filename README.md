# README
## 開発言語
- Ruby3.0.1
- Rails6.1.7
## 実行手順
````````
$ git clone https://github.com/mmnna2024/some-req.git
$ cd some_riq
$ bundle install
$ yarn install
$ rails db:create && rails db:migrate
$ bin/webpeck
$ rails s
````````
